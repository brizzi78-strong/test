# Normalized page layouts

Every illustration fitted onto a consistent 4:5 (8x10-inch) portrait page
canvas with a cream mat, centered and uncropped. Generated from the
originals in the parent folder, which remain untouched.

Note: pixel dimensions match the source art, so these are layout guides,
not print-resolution files. For print, regenerate art at higher resolution
(roughly 2400x3000 px for 8x10 at 300 DPI) and re-run the normalization.
