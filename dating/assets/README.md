# dating/assets

Drop image assets for the Cardinal demo here.

## founders.jpg — the founder couple photo (Hope & founder)

The landing's **"Our why"** section shows a photo from `assets/founders.jpg`.
Add your photo here with that exact name and it appears automatically (the
rose-gradient you see is just the fallback until the file exists).

- Recommended: a portrait-ish crop (~4:5), reasonably sized (e.g. 800–1200px
  tall), JPG or PNG.
- On GitHub Pages / a real web host this "just works" once the file is committed.
- For the **published Artifact** (claude.ai) the image must be *embedded*
  (the artifact can't load external files) — send Claude the actual file and it
  can inline it as a data URI.
